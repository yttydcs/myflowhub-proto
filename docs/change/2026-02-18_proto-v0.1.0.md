# 2026-02-18 发布 v0.1.0（Proto）

## 变更背景 / 目标
Proto 是 MyFlowHub 的“协议字典”仓库（types/常量/协议号等），用于被 Server/SDK/Win 复用，降低跨端重复实现与耦合。

本次发布目标：
- 发布并推送 `github.com/yttydcs/myflowhub-proto@v0.1.0`
- 作为上游仓库移除本地 `replace ../MyFlowHub-Proto` 的可拉取依赖基线

## 具体变更内容
### 新增 / 修改
- 首次发布：将协议定义迁移到独立 Go module（`github.com/yttydcs/myflowhub-proto`）。
- 提供协议定义包（仅协议字段/常量/struct，不包含运行时逻辑）：
  - `protocol/auth`
  - `protocol/exec`
  - `protocol/file`
  - `protocol/flow`
  - `protocol/management`
  - `protocol/topicbus`
  - `protocol/varstore`

### 约束（保证）
- 仅使用标准库；不引入 Core/SDK/Server 依赖。
- 不引入 handler / 网络 / 运行时逻辑；保持“纯协议定义”定位。
